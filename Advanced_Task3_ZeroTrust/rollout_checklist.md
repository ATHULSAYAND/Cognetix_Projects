# Rollout Checklist & Roadmap

Phase 0 — Preparation: Inventory, tagging
Phase 1 — Identity hardening (0–4 weeks): enforce MFA, SSO, CA baseline
Phase 2 — Device posture (4–8 weeks): deploy MDM, EDR
Phase 3 — Micro-segmentation (8–16 weeks): NSG/SG, ingress gateways
Phase 4 — Continuous auth & automation (12–20 weeks): JIT, short-lived creds
Phase 5 — Monitoring & enforcement: SIEM, runbooks, SOAR
KPIs:
- % privileged with MFA
- % services behind mesh
- MTTD, MTTC
